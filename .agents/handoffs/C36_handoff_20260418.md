## 5. Client Intake (ED) System — Full Flow

```
Prospect clicks shareable link:
  https://theaumengine.web.app/#ed-disclosure?ref={advisorUID}
         ↓
  pageEdDisclosure() — consent form (ED = Economic Disclosure)
         ↓ (advisor or prospect completes 10 questions)
  pageEdIntake() — multi-step intake via ed_intake_engine.js
         ↓
  Saved to Firestore: ed_situations/{docId}
  Fields: fullName, wealthTier, lifeStage, profession[], situationScore,
          referringAdvisorUid, assignedAdvisorUid, status, brief, savedAt
         ↓
  Advisor sees submission in:
    A) Client Intake inbox (sidebar → "Client Intake (ED)")
    B) Command Center → Al Briefs section (auto-loaded from _edSituations)
```

### Inbox Buttons
- **Generate Brief →** / **View Brief →**: calls `openEdBrief(id)` → runs `alGenerateBrief()` → navigates to CC → scrolls + amber-pulses `#al-briefs-section`
- **View in CC**: calls `viewEdInCC(id)` → sets `_alActiveSituationId` → navigates to CC → scrolls to brief panel
- **+ New Intake (Self)**: CC header button → `ed-disclosure` (advisor enters manually)
- **Copy Link**: `navigator.clipboard.writeText()` → copies shareable intake URL

### Firestore Read Gate
```javascript
ed_situations read rule:
  allow if isOperator()
  OR request.auth.uid == referringAdvisorUid
  OR request.auth.uid == assignedAdvisorUid
```
> ⚠️ Docs without `referringAdvisorUid` are invisible to non-operators. All 6 current docs were backfilled with `FvEWqsET` this session.

## 7. Recent Cohorts — Wiring

**Location:** Prospect Mine page → right column → "Recent Cohorts" section

**Data source chain:**
```
scripts/write_pipeline_meta.js (run after each Alfred ingest)
  → reads master_leads, counts per nicheId, gets latestIngest date
  → writes to meta/pipeline_stats.nicheBreakdown

db.js fetchAdvisorLeadCount()
  → reads meta/pipeline_stats once on login
  → caches to window._firestoreNicheBreakdown

pages.js pageProspectMine()
  → reads window._firestoreNicheBreakdown
  → renders live niche cohort cards (real counts, real dates)
  → "Load →" button calls window.loadCohort(nicheId)

app.js loadCohort(nicheId)
  → sets activeFilters.niche = nicheId
  → navigate('lead-scoreboard')
  → showToast with niche name + emoji
```
