const isOp = user.email === 'kosal@fin-tegration.com';

    .collection('advisor_pool').doc(user.uid).get();

### Gate Behavior
- **Operator (`kosal@fin-tegration.com`):** Always bypasses — immediate cockpit access

### How to Provision a New Advisor

2. Get UID from Firebase Console → Authentication → Users
3. Run provisioning script with their UID + niche config
4. Advisor refreshes → gate passes → cockpit loads with assigned leads
