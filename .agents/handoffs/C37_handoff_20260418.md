## 4. CIK SCRUB IMPLEMENTATION DETAIL

**File:** `js/data.js`  
**Function:** `getDisplayName(p)` — line ~687  
**Trigger:** Only fires if `firstName` contains `(Cik` pattern

```javascript
if (first && /\(Cik\s*\d+\)/i.test(first + ' ' + last)) {
  const co = (p.company || first)
    .replace(/\s*\(Cik\s*\d+\)/gi, '')
    .replace(/\s*\([A-Z]{1,5}\)/g, '')
    .replace(/^Former:\s*/i, '')
    .trim();
  return co || 'Executive (Name Pending)';
}
```

**Does NOT affect:** Any prospect with a normal name. Zero false positives possible.
